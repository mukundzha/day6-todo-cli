#To-Do CLI

Simple. Fast. Organized.

About

To-Do CLI is a lightweight command-line task manager built with Python. It helps users manage their daily tasks directly from the terminal through a clean and minimal interface. This project was created as part of my journey to master Python by building one project every day.

Features

• Add new tasks

• View all saved tasks

• Mark tasks as completed

• Delete tasks

• Interactive command-line interface

• Simple and beginner-friendly codebase

Why I Built This

This project is part of my 28-Day Python Challenge, where I build one Python project every day to improve my programming skills through consistent practice and real-world projects.
